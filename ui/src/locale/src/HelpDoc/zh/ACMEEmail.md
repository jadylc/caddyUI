## ACME 联系邮箱

用于 ACME 证书申请时注册的联系邮箱。证书颁发机构（如 Let's Encrypt、ZeroSSL）会通过此邮箱发送证书过期提醒、账户验证等通知。

**留空会怎样？**

系统会自动生成一个 `noreply-随机字符串@localhost` 邮箱，证书申请仍可正常进行，但你将无法收到来自 CA 的任何通知。

**哪些场景建议填写？**

- 使用 Let's Encrypt、ZeroSSL 等公共 CA 申请证书
- 希望收到证书即将过期的邮件提醒
- 多人维护同一台服务器，方便 CA 联系管理员

**哪些场景可以留空？**

- 使用 Caddy Internal 自签证书
- 测试或临时环境
- 已有域名监控功能覆盖证书到期检查
